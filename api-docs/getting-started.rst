=====================
Getting Started Guide
=====================

Use this Getting Started Guide to create and access your database instances via
Rackspace Cloud Servers. The examples presented in this guide create a MySQL
database instance. You'll learn how to authenticate, send API requests,
and complete the basic operations for using the Cloud Databases API by using
the cURL and the trove client command line tools.

.. note::
		You can also access |apiservice| features from the
		`Rackspace Cloud Control Panel`_.

.. _Rackspace Cloud Control Panel: https://mycloud.rackspace.com
